🎬 TikTok APIs

📥 Tải Video TikTok GET

/api/dowtik

Parameters:

url (required): URL video TikTok cần tải xuống

Example:

/api/dowtik?url=https://www.tiktok.com/@username/video/123456789

🔥 Xu Hướng TikTok GET

/api/seachtik

Lấy danh sách video xu hướng trên TikTok

Example:

/api/seachtik

👤 Xem Feed User TikTok GET

/api/seachusertik

Parameters:

user (required): Tên người dùng TikTok

Example:

/api/seachusertik?user=username

🔍 Tìm Video TikTok GET

/api/seachvidtik

Parameters:

keyword (required): Từ khóa tìm kiếm video

Example:

/api/seachvidtik?keyword=funny

🖼️ Tìm Ảnh TikTok GET

/api/seachpictik

Parameters:

keyword (required): Từ khóa tìm kiếm ảnh

Example:

/api/seachpictik?keyword=nature

📺 YouTube APIs

📥 Tải Video YouTube GET

/api/ytbdown

Parameters:

url (required): URL video YouTube cần tải xuống

Example:

/api/ytbdown?url=https://www.youtube.com/watch?v=VIDEO_ID

🔗 URL Shortener APIs

✂️ Rút Gọn URL POST

/api/rutgon

Parameters:

url (required): URL cần rút gọn

Example:

POST /api/rutgon
Body: {"url": "https://example.com"}

📊 Response Format

Tất cả API trả về dữ liệu dưới dạng JSON:

{ "success": true/false, "data": "...", "message": "..." }

🚀 Cách Thêm API Mới

  1. Thêm endpoint mới vào section phù hợp
  2. Sử dụng cấu trúc API card với đầy đủ thông tin
  3. Đảm bảo có endpoint, method, parameters và example
  4. Thêm mô tả rõ ràng cho từng parameter
  5. Test API trước khi thêm vào documentation

📞 Liên hệ: Nếu có thắc mắc về API, vui lòng liên hệ admin.
